2. Two entry points

Prover — stepwise tactic search

python -m prover.cli --goal '...' --model 'gemini:gemini-3-flash-preview'

One tactic at a time. Beam search + sledgehammer + counter-example tools.

  • OK LLM tactic generation
  • OK Sledgehammer / Quickcheck / Nitpick
  • OK Beam search
  • OK ML reranker (XGBoost / DQN / AWR)
  • OK Premise selection (bi+cross encoder)

Planner — Isar outline + Fill + Repair

python -m planner.cli --mode auto '...'

LLM writes whole-proof skeleton, fills holes, repairs broken parts.

  • OK Outline generation (multi-temp)
  • OK Micro RAG (token priors + hint lexicon)
  • WIP Fill — call prover at sorry
  • WIP CEGIS Repair — staged regen

5. WIP detail — what's broken, what to fix

Fill (graded)

Earliest sorry → extract local subgoal → call stepwise prover → splice replacement.

  • Fix subgoal extraction (print_state parse, multi-subgoal, schematics)
  • Pass context: assumptions + prior haves + retrieved premises
  • Splice with correct indentation, preserve outer proof…qed
  • Top-down deterministic ordering
  • Feed nitpick CE hints to prover sub-call

files: planner/repair_inputs.py, planner/driver.py:_fill_one_hole

CEGIS Repair (graded)

Earliest non-sorry failure → regen smallest enclosing block → escalate.

  1. Stage 1 local have/show block
  2. Stage 2 case-branch / inner proof…qed
  3. Stage 3 whole proof
  • Earliest failure anchoring from FINISHED JSON
  • Per-stage attempt budget + global timeout
  • Re-Fill any new sorry after edit
  • Define "failed" precisely (no-sorry-unverified OR Fill-can't-close)

files: planner/repair.py

6. Improvement topology

Tier S — must-do

  1. Fill works (subgoal + splice + context)
  2. CEGIS Repair stages 1–3 + budgets + re-Fill loop

Tier A — differentiators

  1. Candidate pool + scoring (lines_verified − open_sorrys)
  2. AFP vector RAG (bi-encoder index + cross-encoder rerank)
  3. Error-typed prompts (type / unknown / unsolved / syntax)

Tier B — polish

  1. Memoize / dedupe proposals
  2. Reranker picks first repair stage
  3. Beam over candidate pool
  4. Local-have harvesting for Fill
  5. Offline retraining of rerankers
